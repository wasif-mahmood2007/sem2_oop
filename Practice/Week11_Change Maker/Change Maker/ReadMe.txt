Cashier–Customer Change Maker (JAVA):
This is a simple Java project that models a cashier–customer interaction for paying bills and receiving change.

Project Structure:
Person.java [Base class for people for cashier or customer name]
Cashier.java [Calculates bills and generates change]
Customer.java [Pays bills and requests change]
Main.java [Demo program to test all features]


Features:
Create customers and cashiers using constructors.
Calculate the balance after bill payment.
Generate change using standard denominations (5000, 1000, 500, …, 1).
Generate change using preferred notes (e.g., only 100s and smaller).
Pay bills and automatically get change.
Override toString() methods to display meaningful details.
