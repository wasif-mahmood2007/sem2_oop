Learning Management System (JAVA):
This project implements an n-dimensional vector class in Java.

Project Structure:
Vector.java [Implements the vector class]
Main.java [Demo program to test all features]

Features:
Create vectors using a default constructor or a parameterized constructor.
Add new coordinates.
Update (set) coordinates.
Remove coordinates (by value or index).
Find a coordinate inside the vector.
Get the dimension of the vector.
Compute the magnitude of a vector.
Addition of two vectors.
Subtraction of two vectors.
Dot product of two vectors.
Cross product of two 3D vectors.
Clear all coordinates in a vector.
Display vectors in a readable format using toString().
